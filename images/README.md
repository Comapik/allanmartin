# images/

Placez vos images dans ce dossier en respectant la convention de nommage suivante :

## Couvertures de séries (utilisées dans la grille d'accueil)
- `series/serie-01-cover.jpg`
- `series/serie-02-cover.jpg`
- …

## Photos d'une série (utilisées dans la galerie de la page série)
- `series/serie-01-01.jpg`
- `series/serie-01-02.jpg`
- …

## Formats recommandés
| Usage              | Format | Taille conseillée |
|--------------------|--------|-------------------|
| Couverture série   | JPG    | 1600 × 1000 px    |
| Photo série        | JPG    | 2000 px (côté long) |
| Hero pleine page   | JPG    | 2400 × 1600 px    |

Optimisez vos images avec [Squoosh](https://squoosh.app/) ou ImageOptim avant de les déployer.
